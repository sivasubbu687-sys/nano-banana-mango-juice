# Nano Banana 🍌

An elegant, premium, and fully responsive scrollytelling web application for **Nano Banana** cold-pressed juices.

## 🚀 Live Site
Your live website is permanently deployed at:
👉 **[https://nano-banana-juice.netlify.app/](https://nano-banana-juice.netlify.app/)**

---

## Features
- **Modern Scrollytelling Experience**: Engaging scroll-linked animations powered by Framer Motion.
- **Premium Design System**: Tailored gradients, typography, and glassmorphism layouts.
- **Flavors Included**: Cream Mango, Dutch Chocolate, and Ruby Pomegranate.
- **Automated Deployment**: Fully configured for continuous integration with Netlify.

---

## Getting Started (Local Development)

First, install the dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to view your local development server.
